Path

File path is the address of a particular file.eg:image we can access a unique address of the file.

Types of file path

  1. Absolutely file path
  2. Relative file path

Absolutely file path

An absolute path is the location of a file or directory specified from the root directory.
It represents the complete path from the start of the filesystem from the root directory.

Example

D:/My/Esho.png is an absolute path. All of the information needed to locate the file is contained in the path string.

Relative file path

A relative file path in HTML refers to the location of a file relative to the current web page's location.

Example

In This example, the relative file path “my/esho.png” indicates that the image file “jesus.
jpg” is located in a subfolder named “my” relative to the current HTML file.
